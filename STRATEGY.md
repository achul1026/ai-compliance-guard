# 📈 STRATEGY — ai-compliance-guard 시장 전략

> 누구에게, 어떤 가치를, 어떻게 팔 것인가에 대한 비즈니스/GTM 전략.
> 개발 순서는 [ROADMAP.md](ROADMAP.md), 서비스 운영은 [OPERATIONS.md](OPERATIONS.md) 참고.

---

## 1. 시장 & 문제 정의

- **문제**: 건강기능식품·화장품의 SNS 광고 카피는 식약처 표시·광고 규제가 엄격하다. 위반 시 행정처분·과징금·게시물 삭제 리스크가 있으나, 셀러·브랜드가 광고 문구마다 법령을 직접 확인하기는 현실적으로 어렵다.
- **시장 동향**: SNS 커머스·인플루언서 마케팅 확대로 광고 카피 생산량 급증, 동시에 규제·단속은 강화되는 추세 → 사전 점검 수요 증가.
- **핵심 가설**: "광고를 올리기 전에 위반 여부를 빠르고 저렴하게 확인하고 싶다"는 페인이 반복적으로 발생한다.

## 2. 타겟 고객 (세그먼트)

| 세그먼트 | 특징 | 수요 강도 |
| :--- | :--- | :--- |
| 마케팅 대행사 / 인플루언서 매니지먼트 | 다수 클라이언트 카피를 반복 검수 | 매우 높음 (헤비 유저) |
| 건강기능식품 브랜드 (중소) | 자율심의 대상, 사전 점검 수요 명확 | 높음 |
| 화장품 셀러/브랜드 (중소) | 자사몰·SNS 카피 직접 작성 | 중간 (간헐적) |

- **1차 타겟 추천**: 페인이 가장 빈번하고 **반복 수요**가 큰 마케팅 대행사 또는 건기식 브랜드부터 공략. 화장품 셀러는 확산 단계 타겟.

## 3. 가치 제안 (Value Proposition)

- **사전 예방**: 행정처분·과징금 리스크를 게시 전에 차단 → 법무 비용 절감.
- **Remediation**: 단순 위험 탐지를 넘어 바로 쓸 수 있는 안전한 대체 문구 제공.
- **보안**: 광고 카피·계약서가 유출되지 않는다는 신뢰 — B2B 고객 유치의 핵심.
- **환각 제로**: 비판관 교차검증으로 근거 조항의 신뢰성 확보 → 범용 챗봇과의 결정적 차별점.

## 4. 경쟁 분석

| 대안 | 한계 |
| :--- | :--- |
| 수동 법령 검토 | 느리고 전문성 필요, 누락 위험 |
| 법무 외주/자문 | 비용 높고 카피 단위로 쓰기엔 비현실적 |
| 협회 자율심의 | 사전 절차로 시간 소요, 카피 수정 반복에 부적합 |
| 범용 ChatGPT 등 | 환각·근거 부정확, 최신 규정 미반영, 보안 불안 |

- **차별점**: 도메인 특화 Hybrid RAG + **근거 조항 명시** + 대체 문구 제안 + 데이터 보안.

## 5. 가격 전략

- **Free Tier**: 월 3~5회 무료 검사 — 신규 고객 획득(acquisition) 목적.
- **Paid Tier**: 두 모델 병행 검토.
  - *건당 크레딧 차감*: 간헐적으로 쓰는 셀러에 적합.
  - *월간 구독(SaaS)*: 반복 검수하는 대행사·브랜드에 적합.
- **가격 검증**: [OPERATIONS.md](OPERATIONS.md)의 검사당 원가 위에 마진을 확보하는 가격. 고객 지불의사(WTP)는 인터뷰·베타로 검증.
- **가격 기준**: 절감해 주는 리스크 비용(과징금·삭제·재작업)을 기준으로 한 **가치 기반 가격** 책정.

## 6. GTM / 초기 고객 확보

- **확보 채널**:
  - 셀러·마케터 커뮤니티(네이버 카페, 오픈채팅 등) 침투.
  - 건기식·화장품 업종 협회·세미나 연계.
  - 마케팅 대행사 직접 영업(헤비 유저 우선).
  - 콘텐츠 마케팅: "건기식 광고 이렇게 쓰면 처벌받습니다" 류 규제 가이드 콘텐츠로 유입.
- **초기 전술**: 소수 헤비 유저 대상 무료 베타·파일럿 → 밀착 피드백으로 제품 다듬기.
- **보안 마케팅**: "당신의 카피는 안전하게 보호되며 절대 유출되지 않습니다"를 랜딩 페이지 전면에 배치.

## 7. 검증 마일스톤 (제품-시장 적합성)

| 시점 | 검증 항목 |
| :--- | :--- |
| Phase 1 완료 후 | 검색 정확도 내부 평가 세트 통과 |
| Phase 2 완료 후 | 베타 사용자 대상 판정 품질·유용성 피드백 수집 |
| Phase 3 / 런칭 후 | 유료 전환율, 리텐션, 검사당 원가 대비 매출 |

## 8. 사업 리스크

- **규제 변화**: 규정 개정 대응 부담 → [OPERATIONS.md](OPERATIONS.md) 1번 항목으로 관리.
- **LLM 비용 변동**: 원가 상승 시 마진 압박 → 비용 운영·모델 라우팅으로 대응.
- **경쟁 진입**: 도메인 데이터 품질·신뢰 자산으로 진입장벽 구축.
- **신뢰 사고**: 오판정 1건이 평판에 큰 타격 → 품질 운영·법적 디스클레이머로 방어.
